Feito por: Gabriel Lima

📜 Como Jogar:

Objetivo: Desvie dos obstáculos e colete estrelas para aumentar sua pontuação!

🎮 Controles:

    Setas direcionais: Movimentam o jogador.

    Barra de espaço: Realiza o dash.

⚡ Dash e Cooldown:

O dash permite que você se mova rapidamente e fique invencível por um curto período.

Ele tem um cooldown de 2 segundos, indicado por uma barra acima do jogador:

    Amarela: O dash está recarregando.

    Laranja: O dash está pronto para ser usado!

💡 Dicas:

    ✅ Use o dash para escapar de obstáculos inesperados!

    ✅ Colete estrelas para aumentar sua pontuação!

    ✅ Desvie dos obstáculos para sobreviver o máximo possível!

🚀 Boa sorte!
